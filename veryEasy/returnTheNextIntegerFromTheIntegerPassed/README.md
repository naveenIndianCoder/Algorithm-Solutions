### Return the Next Number from the Integer Passed - Create A Function that Takes:

1. An **Integer** as an **Argument**.
2. **Increments** the **Argument Integer** by **+1** and Shows the **Output** in the **Console**.

> nextNumber(10) ➞ 11

> nextNumber(-4) ➞ -3

> nextNumber(21) ➞ 22
