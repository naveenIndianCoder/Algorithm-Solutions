### Is the Number Even or Odd? - Create A Function that Takes:

1. A **Number** as an **Argument**.
1. Shows **Even** in the **Console**, If the **Argument Number** is **Even**. 
1. Otherwise shows **Odd**.

> evenOrOdd(12) ➞ "Even"

> evenOrOdd(291) ➞ "Odd"

> evenOrOdd(12508) ➞ "Even"
