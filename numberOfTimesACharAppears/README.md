### Number of Times a Character Appears - Create a function that takes: 

1. A **String** and A **Character** as an arguments. 
1. Shows the **Number** of **Times** argument character appears in **Each Word** in the **Argument Sentence**, in the console.

> numTimesCharAppearsInStr("i am a javascript programmer", "a") ➞ [0, 1, 1, 2, 1] 

> numTimesCharAppearsInStr("i live in jaipur, rajasthan", "i") ➞ [1, 1, 1, 1, 0]

> numTimesCharAppearsInStr("i am a self taught programmer", "m") ➞ [0, 1, 0, 0, 0, 2]

Note : Treat **Upper** and **Lower Case Characters** of the **Same Letter** as being **Identical**.
