### Equal to 7 (Fix the Expression) - Create A Function that Takes:  

1. A **Number** as an **Argument**.
1. Shows **true** in the **Console**, If the **Argument Number** is **Equal** to **7**.
1. Otherwise shows **false**.

> equalTo7OrNot(33) ➞ false
