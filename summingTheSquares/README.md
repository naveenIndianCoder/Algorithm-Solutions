### Summing the Squares - Create a function that takes: 

1. An **n (Integer)** as an argument. 
1. Shows the **Sum** of **All Square Numbers Up To** and **Including n** in the console.

> nSquaresSum(14) ➞ 1015 

> nSquaresSum(20) ➞ 2870

> nSquaresSum(23) ➞ 4324
