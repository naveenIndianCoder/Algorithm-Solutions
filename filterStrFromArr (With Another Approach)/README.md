### Filter out Strings from an Array - Create a function that takes: 

1. An **Array** as an argument. 
1. **Filters** out **Strings** from **Argument Array**, and shows a **New Array** containing **Only Integers** in the console.

> filterOutStr([23, "morning", 96, "nineteen", "eleven", 26]) ➞ [23,96,26] 

> filterOutStr([20, "april", "ninety nine", 7, 28, "afternoon"]) ➞ [20,7,28]

> filterOutStr(["house", 7, 12, "area", "jaipur", 302004, 25]) ➞ [7,12,302004,25]
