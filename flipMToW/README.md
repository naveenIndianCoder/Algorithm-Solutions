### Flip M's to W's - Create a function that takes: 

1. A **Uppercase String** as an argument. 
1. **Flips** all **M's** to **W's** of argument string and shows the output in the console.

> flipMToW("MONKEY") ➞ "WONKEY" 

> flipMToW("MAKE UP") ➞ "WAKE UP"

> flipMToW("I AM A WRITER.") ➞ "I AW A WRITER."
