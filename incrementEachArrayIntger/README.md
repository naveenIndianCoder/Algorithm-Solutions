### Increment Each Array Integer - Create a function that takes:

#### Difficulty - Very Easy

1. An **Integer Array** as an argument. 
1. **Increments Every Integer** of **Argument Array** and shows the **New Array** in the console.

> incrementEveryInt([-61, 52, 27, -2221]) ➞ [-60, 53, 28, -2220] 

> incrementEveryInt([52, 113, -287]) ➞ [53, 114, -286]

> incrementEveryInt([-35, 122002]) ➞ [-34, 122003]
